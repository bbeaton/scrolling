Features:
==========
a)HIOX NEWS TICKER enables users to have numerous text links scroll upwards.
b)Text with links can be given in a seperate text file.
c)The width, height and speed of scrolling can be controlled.
d)The text stops scrolling when mouse is moved over.
e)Color and Look and Feel are configurable.
f)Ease of use.


Installation:
=============
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation.

a) Unzip HNT_1_0.zip. to extract the files to the  folder \HNT_1_0\HNT_1_0

File Permissions:
 a) Give read and execute (644) permission for news.txt and colors.txt (Only of
Linux users).

Add/Edit News:
 a) Open the file HNT_1_0/news.txt
 b) Add or edit the news in the same format already given in the file.

Configuring the Look and Feel:
 a) Open the file HNT_1_0/colors.txt.
 b) Change the following code to change the particular fields
      background-color="#887788"
      font-color="#ffffff"
      scrolldelay="40"

Embedding the code:
 a) Open the file HNT_1_0/code.txt in a browser
 b) Copy the code in the text box given.
 c) Paste the code any where needed.
